# Gamification Plan

| Module | Goal | Trigger | Reward | KPI |
|--------|------|---------|--------|-----|
| XP Loop | Increase onboarding completion | Complete profile, add listing | XP + badge | Level attainment % |
| Streak Engine | Boost daily logins | Bookings/check-ins | Bonus XP, coupons | 7-day streak completion % |
| Quest Builder | Guide partners through onboarding | Step-by-step quest | Discount, unlock features | Quest completion rate |

Reference modules from `packages/gamification/` and include supporting metrics in `docs/06-pdr/po-dashboard.md`.
