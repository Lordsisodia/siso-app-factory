# UI Best Practices Research Hub

Use this folder to drop Notebook LM exports, distilled checklists, or benchmark findings related to interface best practices. Suggested structure:

- `heuristics/` – Jakob Nielsen, GOV.UK, Apple HIG notes
- `patterns/` – Examples of flows, navs, dashboards, multi-tenant theming
- `benchmark-data/` – Metrics, A/B results, conversion lift stats
- `references.md` – Citations + links to source studies

AI agents should read this folder before component planning (Phase 5) to pull the freshest guidance.
